Politique de Confidentialité AgregLLM

Dernière mise à jour : 19 janvier 2026

Résumé rapide : AgregLLM est une application conçue pour respecter votre vie privée. Nous n'avons accès à aucune de vos données. Vos conversations sont stockées exclusivement sur votre navigateur local et sur votre propre espace Google Drive personnel.

1. Collecte des données

AgregLLM collecte uniquement les informations que vous choisissez de sauvegarder via l'extension de navigateur :

  • Titres des discussions
  • URL des conversations
  • Noms des services LLM utilisés (ex: ChatGPT, Claude)
  • Résumés sommaires des discussions
  • Tags et dossiers d'organisation créés par l'utilisateur

2. Utilisation des données (Services Google)

AgregLLM utilise l'API Google Drive avec le champ d'application (scope) drive.appdata.

Cette autorisation permet à l'application de :

  • Créer et mettre à jour un fichier de sauvegarde nommé agregllm_backup.json.
  • Ce fichier est stocké dans un dossier caché spécial sur votre Google Drive, dédié uniquement à l'application.
  • AgregLLM n'a pas accès à vos autres fichiers, photos ou documents Google Drive.

3. Stockage et Sécurité

Toutes les données sont traitées localement dans votre navigateur. Aucune donnée n'est envoyée vers nos serveurs ou vers des tiers, à l'exception de la synchronisation directe vers votre compte Google Drive personnel via une connexion sécurisée (HTTPS).

4. Partage des données

Nous ne vendons, n'échangeons ni ne transférons aucune information à des tiers. Vos données vous appartiennent.

5. Vos droits

À tout moment, vous pouvez :

  • Supprimer vos données locales via les paramètres de l'application.
  • Supprimer votre sauvegarde Cloud directement depuis l'application ou en gérant les applications tierces dans votre compte Google.
  • Exporter vos données au format JSON.

6. Contact

Pour toute question concernant cette politique, vous pouvez contacter le développeur via le dépôt GitHub du projet : https://github.com/Fredb34670/AgregLLM
